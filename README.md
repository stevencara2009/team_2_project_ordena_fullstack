# Ordena
El sistema ORDENA está diseñado para optimizar la gestión operativa del restaurante de la empresa ALIMENTOS ANDINOS S.A., permitiendo el control integral de pedidos, trazabilidad de ventas, administración de usuarios y disponibilidad de mesas en tiempo real. Su implementación busca mejorar la eficiencia del servicio, reducir errores humanos y facilitar la toma de decisiones mediante reportes automatizados. ORDENA funcionará como una aplicación web accesible desde la web y responsiva para dispositivos móviles, principalmente tablets..


## 🚀 Cómo ejecutar el proyecto

Para ejecutar este proyecto localmente, siga estos pasos:

1. **Instalar dependencias**:
   Asegúrese de estar en la carpeta raíz y ejecute:
   ```bash
   npm install
   ```

2. **Ejecutar en modo desarrollo**:
   Una vez instaladas las librerías, inicie el servidor con:
   ```bash
   npm run dev
   ```

3. **Ver la aplicación**:
   Abra su navegador en la dirección que indique la terminal (normalmente `http://localhost:5173`).
